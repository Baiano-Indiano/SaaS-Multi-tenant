# Roadmap

This project follows a phased High-level functional requirements for **Milestone v3.0 (Enterprise & Scale)**.

### R-13: Enterprise Landing Page
- Expand page content with "Benefits", "Pricing" and "FAQ" sections.
- Use GSAP ScrollTrigger for all section reveals.
- Implement magnetic hover effects on main CTA buttons.

### R-14: Data Visualization
- Create 2+ new interactive chart types (Area, Pie) using SVG + GSAP.
- Implement "Staggered Entry" for all dashboard card elements.
- Add real-time status indicators with pulse animations.

### R-15: Connectivity Ecosystem
- CRUD for API Keys with scope management per organization.
- Webhook registration system (URL, Secret, Events).
- API Key validation middleware for `/api/v1/*` routes.

### R-16: Core Automations
- Simple workflow builder UI.
- Background delivery service for webhooks.
- Event-based trigger system (e.g., "On Member Joined").

### R-17: Member Management
- Data table for displaying organization members.
- Interactive modal for inviting new members via email.
- Integration with Drizzle ORM and existing member schema.
- UI enhancements using shadcn/ui and Framer Motion.
