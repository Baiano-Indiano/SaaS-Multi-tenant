# Requirements: Multi-Tenant SaaS Starter

**Defined:** 2026-04-16
**Core Value:** Secure, tenant-isolated data architecture with flexible organization management that accelerates the launch of enterprise-ready B2B applications.

## v1 Requirements

### Authentication & Core

- [x] **AUTH-01**: User can sign up with email and password
- [x] **AUTH-02**: User session persists across browser refresh (Better-Auth)
- [x] **AUTH-03**: User profile management (name, email update)

### Organization Setup

- [x] **ORG-01**: User can create a new organization
- [x] **ORG-02**: Organization creation cascades to create a new dedicated schema in PostgreSQL
- [x] **ORG-03**: User can switch active organization context without re-authenticating

### Member Invitations

- [x] **INV-01**: Organization Admin can generate an invitation link for a new member
- [x] **INV-02**: Invitations are bound to a specific PostgreSQL schema / Organization
- [x] **INV-03**: User can accept invitation to join the organization

### Access Control (RBAC)

- [x] **RBAC-01**: Organization Admin can create custom Roles within their organization
- [x] **RBAC-02**: Admin can assign specific Permissions to custom Roles
- [x] **RBAC-03**: Admin can assign Roles to organization Members
- [x] **RBAC-04**: Edge/Middleware routes protect data based on Active Organization context and Permissions

### Frontend Experience

- [x] **UI-01**: High-impact marketing Landing Page with Anime.js hero animations
- [x] **UI-02**: App dashboard utilizes Framer Motion for state changes
- [x] **UI-03**: Theme architecture utilizing Tailwind CSS v4 and shadcn/ui

### Billing & Subscriptions

- [x] **BILL-01**: Stripe integration for organization subscription plans
- [x] **BILL-02**: Plan-based restriction on member limits

### Project Management

- [x] **PROJ-01**: Create and manage isolated business projects within a specific tenant context


## v2 Requirements

### Advanced Notifications
- **NOTF-01**: In-app WebSocket notification alerts

## Out of Scope

| Feature | Reason |
|---------|--------|
| Custom Domain Routing | `tenant.saas.com` adds too much Edge middleware and DNS complexity for a starter. Path-based routing `/org/[orgSlug]` is used. |
| Anime.js for App UI | Fights React VDOM and lifecycle hooks causing UI sync bugs. |
