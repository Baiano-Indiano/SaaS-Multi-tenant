# Plan 39-02: Frontend Integrations UI, Localization & Setup Drawers

**Goal:** Create a premium integrations dashboard settings page supporting Slack (OAuth), Microsoft Teams (OAuth & Webhook Fallback), and Custom Webhooks with dedicated modal dialogs, status badges, and smooth Framer Motion/TVA transitions.

<threat_model>
- **Threat 1:** CSRF or state hijacking during the client redirect flow.
  - *Mitigation:* The frontend redirect routes explicitly request server-side state tokens and pass them to authorization redirects.
- **Threat 2:** Input validation evasion (e.g. malformed Webhook URLs, local network SSRF attempts).
  - *Mitigation:* Webhook inputs are validated using Zod on the client-side and sanitization/SSRF filters are applied in Server Actions.
</threat_model>
